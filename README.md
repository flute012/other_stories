# 課程學習活動網站UI設計展示
![專案示意圖](https://flute012.github.io/other_stories/draft.png)
**展示：** https://flute012.github.io/other_stories/
### 解決方案

將三層資訊整合為**單一介面**，使用者可以在同一個畫面中：
- ✅ 切換冊次
- ✅ 導覽課次
- ✅ 直接選擇功能
- ✅ 功能位置不變，可以形成使用記憶

**目標：** 在目錄頁即可找到目標功能，大幅降低操作路徑，減少用戶網頁載入、選擇負擔。

---

## ✨ 功能特色

### 1. 三維導覽系統
- **冊次切換**：左側垂直列表，支援書冊切換
- **課次導覽**：頂部顯示當前與相鄰課次，支援左右滑動與快速跳轉
- **功能選擇**：功能按鈕，清晰圖示作為視覺分類

### 2. 多種互動方式
- **觸控滑動**：左右滑動切換課次（支援手機）
- **快速導航點**：4 個圓點，每點代表 3 課，支援快速切換
- **課次標籤**：直接點擊課次圓圈切換


## 🎨 設計理念

### 1. 減少認知負擔
- **一致的視覺語言**：圓形元素貫穿整個介面（課次、導航點、功能圖示）
- **色彩編碼**：8 個功能用不同的漸層色區分

### 2. 提升操作效率
- **快捷方式**：導航點可跳過 3 課，減少滑動距離
- **即時回饋**：冊次、課次點選都有視覺回饋，底圖變化。

### 3. 資料結構
**使用階層式結構**：

booksData = [
  {
    id: 1,
    name: '第一冊',
    lessons: [...]  // 12 課
  }
]


**優點**：
- 易於擴充（增加冊次或課次）
- 切換冊次時可完整載入該冊資料
